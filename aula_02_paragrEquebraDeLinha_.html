Parágrafos e quebra de linhas.

O parágrafo pode ser escrito de qualquer maneira. Desde que esteja dentro das tags. <p> e </p>

Se precisar quebrar o texto em algum lugar específico
Você pode usar a tag BR

Para mostrar uma tag sem interferir no código basta colocar "& lt;", tudo junto, para o sinal de menor "<" e "& gt;", tudo junto, para o sinal de maior ">"
